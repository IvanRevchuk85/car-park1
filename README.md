# CarPark

Automotive marketplace platform built by IRV LABS.

---

## Overview

CarPark is a mobile-first automotive marketplace platform designed for dealerships and vehicle sales businesses.

The platform helps dealerships manage inventory, publish vehicle listings, generate leads and provide customers with a modern mobile browsing experience.

---

## Key Features

### Customer Application

- Vehicle catalog
- Vehicle details
- Vehicle photo gallery
- Advanced search and filters
- Favorites
- Dealership information
- Lead generation
- Contact dealership directly

### Administration Panel

- Vehicle management
- Photo management
- Inventory control
- Lead management
- Pricing management
- Publishing workflow

## Deployment

The backend is deployed on AWS and the mobile application is connected to the live server environment.

The project is not just a static prototype. It includes a working backend, database, media handling and production-oriented mobile client integration.
